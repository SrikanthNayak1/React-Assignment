This project is a React-based web application featuring a counter, user data form, and rich text editors, along with smooth animations, data persistence, and routing.

Features

Counter Component:

Increment, decrement, and reset buttons.

Background color height changes with count.

Reset button resets background to default.

Smooth animations using React Spring.

User Data Form:

Fields: Name, Address, Email, Phone.

Autogenerated User ID.

Data persistence using local storage.

Warning popup for unsaved changes before closing the browser.

Rich Text Editor:

Visualize user data.

Text formatting (bold, italic, underline, lists).

Dashboard (Optional):

Visual representation of user profile trends using React Charts.

Routing:

React Router for navigation between Counter, Form, and Dashboard.

Tech Stack

React + TypeScript

MUI / Chakra UI / Next UI (Optional)

React Spring (Animations)

React Router

Draft.js (Rich Text Editor)

React Charts (Dashboard)

Redux Toolkit (Optional)

Installation

npm create vite@latest react-app
cd react-app
npm install
npm install @mui/material @mui/icons-material @emotion/react @emotion/styled
npm install @chakra-ui/react @chakra-ui/icons @emotion/react @emotion/styled framer-motion
npm install @react-spring/web
npm install react-router-dom
npm install draft-js react-draft-wysiwyg
npm install react-redux @reduxjs/toolkit
npm install react-chartjs-2 chart.js

Running the App

npm run dev
